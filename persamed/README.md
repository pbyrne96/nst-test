

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

.ENV includes the following so far
        ```ini
        POSTGRES_URL=''
        GRAPH_DB_URL=''
        AWS_API_VERSION='latest'
        AWS_REGION='eu-west-1'
        ```
